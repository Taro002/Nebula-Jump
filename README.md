**Nebula Jump** is an intense, neon-styled arcade platformer. Navigate through a procedurally generated galaxy, blast through obstacles, and climb the scoreboard!

---

## 🚀 How to Play

### ⬇️ Direct Play (Recommended)
You don't need to install Python to play!
1. Go to the **[Releases](https://github.com/Taro002/Nebula-Jump/releases)** section.
2. Download the latest `NebulaJump.exe`.
3. Run the file and start jumping!

### 💻 For Developers
If you want to explore or modify the source code:
1. Clone the repo: `git clone https://github.com/Taro002/Nebula-Jump.git`
2. Install dependencies: `pip install pygame`
3. Launch the game: `python mg.py`

---

## 🎮 Controls

| Key | Action |
| :--- | :--- |
| **Space** | Jump / Double Jump |
| **Left / Right Arrows** | Move Character |
| **X** | Shoot Projectiles |
| **Enter** | Start from Menu |
| **R** | Restart after Game Over |
| **C** | Cycle Player Colors |

---

## ✨ Key Features

* 🛠️ **Procedural Generation**: Every level is unique, providing a new challenge every time.
* 🔫 **Triple Shot Power-up**: Find the golden bonuses to unleash triple firepower.
* 💥 **Destructible Walls**: Use your laser to break through orange barriers.
* 🔊 **Custom Sound Engine**: Real-time synthesized audio for a retro sci-fi feel.
* 🌈 **Customization**: Choose your favorite neon skin before starting.
